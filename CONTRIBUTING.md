# 贡献指南 (Contributing Guide)

感谢你对 **Awesome Claude Skills ZH** 的关注！欢迎通过 Pull Request 提交优质资源。

Thanks for your interest in **Awesome Claude Skills ZH**! Contributions via Pull Request are welcome.

---

## 收录标准 (Inclusion Criteria)

提交的资源应满足以下条件：

A submitted resource should meet the following criteria:

1. **相关性 (Relevant)**：与 Claude Skills、Agent Skills 或 AI 智能体开发直接相关。
   Directly related to Claude Skills, Agent Skills, or AI agent development.
2. **可访问 (Accessible)**：链接有效，可正常访问（返回 200），且内容公开可用。
   The link must be valid (returns HTTP 200) and the content publicly accessible.
3. **高质量 (High-quality)**：内容有实质价值，避免重复、过时或纯营销性质的链接。
   Provides substantial value; avoid duplicate, outdated, or purely promotional links.
4. **活跃维护 (Maintained)**：优先收录仍在维护或具有长期参考价值的项目。
   Prefer actively maintained projects or those with lasting reference value.

---

## PR 流程 (Pull Request Process)

1. **Fork** 本仓库并创建特性分支。
   Fork this repository and create a feature branch.
2. 将你的资源添加到 **合适的章节**，并保持与现有条目一致的格式。
   Add your resource to the **appropriate section**, matching the existing entry format.
3. 条目格式 (Entry format)：

   ```markdown
   - [**项目名称 (Name)**](https://example.com)
     一句话简介，说明该资源的核心价值。
   ```

4. 自查链接可访问、无拼写错误、未与现有条目重复。
   Verify the link works, check spelling, and ensure no duplicates.
5. 提交 PR，并在描述中简要说明 **为什么值得收录**。
   Open a PR with a short description of **why it deserves inclusion**.

## 行为准则 (Code of Conduct)

请保持友善与尊重。我们欢迎一切善意的贡献。

Please be kind and respectful. All good-faith contributions are welcome.
